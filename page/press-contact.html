Do you have a question?

If you have a question we encourage you to use the proper communication channels to reach out and we will do our best to assist you!

  • IRC (All channels on Freenode)
  • Support chat for jQuery, jQuery UI: #jquery
  • Support chat for jQuery Mobile: #jquerymobile
  • Development chat for jQuery Core: #jquery-dev
  • Development chat for jQuery UI: #jqueryui-dev
  • Development chat about jQuery Mobile: #jquerymobile-dev
  • Chat for documentation on all projects: #jquery-content
  • Chat for discussion of servers/websites/deployment, etc: #jquery-infrastructure
  • Weekly Meetings for all projects are held in #jquery-meeting, according to the schedule at http://jquery.org/meeting

Found a bug?

For reporting bugs in libraries, documentation, or content, the project’s GitHub issues tracker should be used, except in the case of jQuery Core and jQuery UI, which track issues at http://bugs.jquery.com and http://bugs.jqueryui.com, respectively. The rest of the jQuery Foundation projects can be found at http://github.com/jquery
